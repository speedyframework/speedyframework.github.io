Getting Started

You can get started in a few minutes. Create your project, run and start development. That's all!

Step 1

Create Your Project
Click to the following button. Create your project and download the zip archive.
 
Speedy Initializer

Step 2

Run Your Application
Unzip the zip archive. Open a console and run the following command in the project directory. 

./mvnw spring-boot:run

Step 3

Start Development
Import the project to your IDE (we recommend STS) as a maven project and start development.
